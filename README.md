# Dynamic Home - Flutter JSON UI Engine

A powerful, highly extensible, standalone **JSON-to-Flutter-UI rendering engine**. Build entire Flutter screens or embeddable components dynamically directly from JSON configuration. 

Perfect for Server-Driven UI (SDUI), live A/B testing, no-code/low-code builders, and delivering dynamic content without pushing app updates.

> 🌟 **Cross-Platform Ecosystem:** This package is part of the **Dynamic Home** ecosystem. We provide identical packages for both **Flutter** (`flutter_json_dynamic_home`) and **React Native** (`react-native-json-dynamic-home`). You design your UI once, and the exact same JSON powers both platforms!

## ✨ Advanced Package Features

- **100% Decoupled Architecture:** No hardcoded app dependencies. It natively uses `Theme.of(context)` to adapt to Light/Dark modes instantly.
- **Advanced Caching Engine:** Built-in `JsonLoaderService` handles network requests and local caching via `shared_preferences` to support offline capabilities.
- **Native Action Registry:** Don't just show UI; trigger native Dart code (like purchases, dialogs, analytics) directly from JSON clicks.
- **Conditional Routing:** Support premium vs free user flows by pointing different user cohorts to different JSON configurations via your Flutter app logic.
- **Flexible Embedding:** Use `DynamicScreen` to render full, standalone pages, or `DynamicWidgetArea` to inject a small JSON snippet directly into an existing Flutter view.
- **Built-in Pull-to-Refresh:** Users can automatically refresh dynamic screens just by pulling down.

## 🛠️ Advanced Usage & Example Codes

### 1. Global Setup with Headers
Configure the global network behavior in your `main.dart`. You only need to do this once.

```dart
import 'package:flutter/material.dart';
import 'package:flutter_json_dynamic_home/flutter_json_dynamic_home.dart';

void main() async {
  WidgetsFlutterBinding.ensureInitialized();
  
  // (Optional) Pass API keys or Auth Tokens if your endpoint is secured
  JsonLoaderService.defaultHeaders = {
    'Authorization': 'Bearer YOUR_TOKEN_HERE',
    'x-app-version': '1.0.0'
  };

  runApp(const MyApp());
}
```

### 2. Initialize the Engine with Analytics & Custom Actions
Track every click and screen view dynamically, and link JSON buttons to native code!

```dart
class MyAnalyticsHandler implements JsonAnalyticsDelegate {
  @override
  void logEvent(String eventName, Map<String, dynamic> parameters) {
    print("Analytics Event -> $eventName: $parameters");
  }

  @override
  void logScreenView(String screenName) {
    print("Screen View -> $screenName");
  }
}

final engine = JsonWidgetEngine(
  analyticsDelegate: MyAnalyticsHandler(),
);

// Register Native Actions so the JSON can trigger them!
engine.actionRegistry.register('purchase_premium', (context, params) {
  final plan = params['plan_id'];
  print("Trigger Native Purchase Flow for: $plan");
});

engine.actionRegistry.register('show_toast', (context, params) {
  ScaffoldMessenger.of(context).showSnackBar(
    SnackBar(content: Text(params['message'] ?? 'Action triggered!')),
  );
});
```

### 3. Displaying Full Pages (`DynamicScreen`)

```dart
Navigator.push(
  context,
  MaterialPageRoute(
    builder: (_) => DynamicScreen(
      // Pass the URL generated by the Dynamic Home Screen Editor
      jsonFile: 'https://json.dynamichome.in/api/screens/custom_onboarding.json', 
      engine: engine,
      
      // Advanced: Custom Loading state
      loadingWidget: const Scaffold(
         body: Center(child: CircularProgressIndicator(color: Colors.purple)),
      ),
      
      // Advanced: Custom Error state
      errorBuilder: (context, error, onRetry) {
        return Scaffold(
          body: Center(
            child: Column(
              mainAxisAlignment: MainAxisAlignment.center,
              children: [
                const Text("Connection Failed"),
                ElevatedButton(onPressed: onRetry, child: const Text("Retry"))
              ]
            )
          )
        );
      }
    ),
  ),
);
```

### 4. Embedding Components Inline (`DynamicWidgetArea`)

Perfect for promotional banners inside a statically built feed!

```dart
class HomeScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Home Feed')),
      body: ListView(
        children: [
          const ListTile(title: Text('Static App Item 1')),
          const ListTile(title: Text('Static App Item 2')),
          
          // Inject dynamic JSON UI seamlessly directly into a normal screen
          const DynamicWidgetArea(
            jsonFile: 'https://json.dynamichome.in/api/screens/summer_promo_banner.json',
          ),
          
          const ListTile(title: Text('Static App Item 3')),
        ],
      ),
    );
  }
}
```

### 5. Dynamic A/B Testing & User Routing

Control which JSON URL to load based on the user's local state.

```dart
String determineScreenLayout(User currentUser) {
  // Free vs Premium experience
  if (currentUser.isPremium) {
    return 'https://json.dynamichome.in/api/screens/dashboard_premium.json';
  } else {
    return 'https://json.dynamichome.in/api/screens/dashboard_free.json';
  }
}

// Push to dynamic screen
DynamicScreen(
  jsonFile: determineScreenLayout(currentUser),
  engine: engine,
)
```
